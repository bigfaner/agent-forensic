# 2.5: Remove duplicate code from app.go

## Description

Remove 5 functions duplicated from `stats/stats.go` in `internal/model/app.go`. Replace calls with the now-public `stats.ExtractFilePath()`, `stats.ExtractToolCommand()`, `stats.BuildHookDetail()`, `stats.ParseHookMarker()`.

Also consolidate the two `computeSubAgentStats` variants: keep `computeSubAgentStatsFromTurns` (the full version with hook detection) and have `computeSubAgentStats` (children-only) delegate to stats package.
