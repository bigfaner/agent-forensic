# 2.4: Build SubAgent Statistics View (UF-4)

## Description

Build the SubAgent statistics view component for the Detail panel (UI Function 4). When a SubAgent child node is selected, Detail shows a statistics summary: tool call counts, file operations, and duration stats. Support Tab toggle between stats view and individual tool detail view.

## Reference Files

- `ui/ui-design.md` — Component: SubAgent Statistics in Detail (UF-4)
- `ui/prototype/detail-fileops.html` — interactive prototype
- `design/tech-design.md` — Integration 3: SubAgent Stats View → Detail Panel
- `internal/model/detail.go` — existing detail panel

## Acceptance Criteria

- [ ] "subagent stats:" section label in bright cyan
- [ ] "tools:" sub-block: tool name + count + duration (same format as Turn Overview)
- [ ] "files:" sub-block: file list with R×N/E×N (same format as UF-3)
- [ ] "duration:" sub-block: "avg Xs, peak {tool} ({duration})"
- [ ] Tab toggle switches between stats view and individual tool detail view
- [ ] Stats view is the default when selecting a SubAgent child node
- [ ] Unit tests: stats view rendered with mock SubAgentStats, Tab toggle switches view

## User Stories

- **Story 4** (SubAgent): "Given 我选中一个展开的 SubAgent 节点, When Detail 面板展示 SubAgent 统计, Then 统计信息中包含该 SubAgent 操作的文件列表（路径截断至 40 字符，按操作次数降序，最多 20 条）"

## Implementation Notes

- This is the BUILD task — create the rendering function only
- Do NOT wire into `SetEntry()` mode switching yet (that's the integration task 3.3)
- Reuse the same file list rendering helper from UF-3 if possible
- Duration format: "avg 1.9s, peak Bash go test (5.2s)"
- The stats view and tool detail view are mutually exclusive; track state with a field in the detail model
