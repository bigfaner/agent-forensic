# 3.5: Integrate Hook Analysis Panel into Dashboard (UF-6)

## Description

Replace the existing Hook list in Dashboard's Custom Tools block with the enhanced Hook Analysis panel (built in 2.6). Wire Hook Statistics and Hook Timeline sections into the Dashboard, sourced from `SessionStats.HookDetails`.

## Reference Files

- `design/tech-design.md` — Integration 5: Hook Analysis Panel → Dashboard
- `ui/ui-design.md` — Component: Dashboard Hook Analysis Panel (UF-6) Placement
- `internal/model/dashboard_custom_tools.go` — target file, insertion point: replace existing Hook column
- `internal/model/dashboard.go` — Dashboard View() (may need adjustment)

## Acceptance Criteria

- [ ] Existing Hook list in Custom Tools replaced with enhanced Hook Statistics section
- [ ] Hook Statistics shows `HookType::Target ×N` grouping (or `HookType` when target extraction failed)
- [ ] Hook Timeline section added below Statistics
- [ ] Timeline shows color-coded markers per Turn (green/cyan/yellow/magenta by hook type)
- [ ] Both sections hidden when no hooks in session
- [ ] `Tab` in Dashboard cycles focus to Hook Analysis section (header highlighted cyan)
- [ ] Data sourced from `SessionStats.HookDetails`
- [ ] Existing Dashboard behavior otherwise unchanged

## User Stories

- **Story 5** (stats integration): Dashboard shows enhanced Hook statistics with Type::Target grouping
- **Story 5** (timeline integration): Dashboard shows Hook timeline with per-Turn markers
- **Story 5** (empty integration): No Hook panels shown when session has no hooks

## Implementation Notes

- Target file: `internal/model/dashboard_custom_tools.go`
- This is a BREAKING task: replaces existing Hook column rendering
- Insertion point: the section that currently renders hook counts by type only
- Data source: `SessionStats.HookDetails` (pre-computed by stats layer)
- Must preserve backward compatibility for sessions without hooks (empty HookDetails → skip)
- Ensure the new rendering doesn't break existing Dashboard layout for sessions without hook data
