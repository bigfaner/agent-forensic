# 2.2: Build SubAgent Full-Screen Overlay (UF-2)

## Description

Build the SubAgent full-screen overlay component (UI Function 2). Create `SubAgentOverlayModel` as a bubbletea.Model implementing a centered overlay (80% x 90% screen) with three sections: Tool Statistics, File Operations, Duration Distribution. Implement Tab cycling between sections, j/k scrolling within focused section, and all states (loading, populated, empty, error).
