## Implementation Notes

- SubAgent JSONL files are at `filepath.Join(filepath.Dir(sessionPath), "subagents")/*.jsonl`
- Association: SubAgent tool_use entry's Input JSON `agent_id` field → `subagents/{agent_id}.jsonl`
- `ScanSubagentsDir` lists all `*.jsonl` in the subagents directory (caller matches by agent_id)
- `ParseSubAgent` can reuse `ParseSession` logic since JSONL format is the same
- `maxLines` parameter supports degradation for large files (>50k lines)
