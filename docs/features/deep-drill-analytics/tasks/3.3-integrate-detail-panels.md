# 3.3: Integrate File & Stats Views into Detail Panel (UF-3 + UF-4)

## Description

Wire the Turn File Operations (UF-3) and SubAgent Statistics View (UF-4) components into the Detail panel. Modify `buildTurnOverview()` to include the "files:" section, and modify `SetEntry()` to support SubAgent statistics mode with Tab toggle.

## Reference Files

- `design/tech-design.md` — Integration 2: Turn Overview File List → Detail Panel
- `design/tech-design.md` — Integration 3: SubAgent Stats View → Detail Panel
- `ui/ui-design.md` — Component: Turn Overview File Operations (UF-3) Placement
- `ui/ui-design.md` — Component: SubAgent Statistics in Detail (UF-4) Placement
- `internal/model/detail.go` — target file, insertion points: `buildTurnOverview()` and `SetEntry()`

## Acceptance Criteria

- [ ] `buildTurnOverview()` includes "files:" section after "tools:" block, before anomalies
- [ ] Files section calls `stats.ExtractFilePaths(turn.Entries)` for data
- [ ] Files section hidden when turn has no Read/Write/Edit calls
- [ ] `SetEntry()` detects SubAgent child selection and switches to SubAgent stats view mode
- [ ] SubAgent stats view shows tool stats + file list + duration summary
- [ ] `Tab` toggles between SubAgent stats view and individual tool detail view
- [ ] Stats view is default when selecting SubAgent child; tool detail after Tab
- [ ] Both UF-3 and UF-4 components visible at correct positions in Detail panel
- [ ] Existing Detail panel behavior unchanged for non-SubAgent entries

## User Stories

- **Story 4** (Turn files integration): Turn Overview shows file list in Detail panel
- **Story 4** (SubAgent stats integration): SubAgent child selection shows stats with Tab toggle
- **Story 4** (empty integration): Empty cases correctly hide file/stats sections

## Implementation Notes

- Target file: `internal/model/detail.go`
- UF-3 insertion point: `buildTurnOverview()` method, after "tools:" block, before "anomalies:"
- UF-4 insertion point: `SetEntry()` method, detect SubAgent child node (visibleNode.depth == 2)
- Data sources: `ExtractFilePaths(turn.Entries)` for UF-3, `SubAgentStats` from session stats for UF-4
- Add a field to the Detail model to track current view mode (turn-overview / subagent-stats / tool-detail)
- Tab handler: when in subagent-stats mode, switch to tool-detail; when in tool-detail for SubAgent child, switch back
