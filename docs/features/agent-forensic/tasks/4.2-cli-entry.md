# 4.2: CLI Entry Point

## Description

Implement the CLI entry point using Cobra for command/flag parsing. Handles `--lang` flag for language selection and starts the Bubble Tea program. Includes `~/.claude/` directory validation and graceful error handling.

## Reference Files

- `design/tech-design.md` — Architecture > CLI Entry (main.go)
- `design/tech-design.md` — Architecture > Dependencies (cobra)
- `prd/prd-spec.md` — 异常情况处理 (~/.claude/ 不存在 → 显示错误提示并退出)

## Acceptance Criteria

- [ ] `cmd/root.go` implements Cobra root command with `--lang` flag (default "zh", accepts "zh"/"en")
- [ ] `main.go` calls root command
- [ ] `~/.claude/` directory existence check: missing → print error and exit code 1
- [ ] `~/.claude/` permission check: not readable → print error and exit code 1
- [ ] Language flag passed to i18n system before UI init
- [ ] Bubble Tea program started with `tea.NewProgram(appModel, tea.WithAltScreen())`
- [ ] CLI integration tests: flag parsing, missing dir error, permission error
- [ ] `go build` produces single binary `agent-forensic`
- [ ] Test coverage >= 80%

## User Stories

- Story 8 (边界条件): AC5 — dir not found shows error and exits
- i18n: `--lang en` flag support

## Implementation Notes

- Use `os.UserHomeDir()` to construct `~/.claude/` path
- Exit codes: 0 = normal quit, 1 = fatal error
- Alt screen mode for clean terminal restoration
- CLI help text includes keyboard shortcuts summary
