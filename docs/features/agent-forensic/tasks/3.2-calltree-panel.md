# 3.2: Call Tree Panel Model

## Description

Implement the Bubble Tea model for the Call Tree Panel (right panel, 75% width, upper 67%). Displays session call hierarchy with Turn nodes, tool call children, anomaly highlighting, and real-time node insertion. Supports expand/collapse, Turn navigation (n/p), diagnosis trigger (d), and monitoring toggle (m).

## User Stories

- Story 1 (浏览会话调用树): AC1 — right panel displays call tree; AC2 — session change refreshes tree; AC3 — expand/collapse nodes
- Story 2 (定位异常操作): AC1 — slow calls marked yellow; AC2 — unauthorized marked red
- Story 5 (回放历史会话): AC1 — n key next Turn; AC2 — p key previous Turn; AC3 — slow steps highlighted
- Story 6 (实时监听当前会话): AC1 — new nodes within 2s; AC2 — visual flash 3s
- Story 8 (边界条件): AC1 — exactly 30s marked; AC6 — empty file shows empty state

## Implementation Notes

- Flatten tree into a visible-node list for cursor navigation
- Each visible node tracks: depth, TurnEntry pointer, expanded state, anomaly
- Use `lipgloss.NewStyle().Foreground(...)` for ANSI color rendering
- 3-second flash: use `tea.Tick` with timer, track flash end time per node
- Monitoring state: `bool` flag, passed to/from App Model
- Tab when no node selected: auto-select first visible node
- Tab when tree is empty (Loading/Empty/Error): no-op
