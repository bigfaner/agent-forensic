# 2.2: 新增 i18n 翻译键

## Description

在 `internal/i18n/locales/zh.yaml` 和 `en.yaml` 中新增「自定义工具」区块所需的翻译键。

## Reference Files

- `internal/i18n/locales/zh.yaml` — 现有中文翻译
- `internal/i18n/locales/en.yaml` — 现有英文翻译
- `docs/features/dashboard-custom-tools/ui/ui-design.md` — 区块标题和列标题

## Acceptance Criteria

- [ ] zh.yaml 新增以下键：
  - `dashboard.custom_tools: "自定义工具"`
  - `dashboard.custom_tools.skill: "Skill"`
  - `dashboard.custom_tools.mcp: "MCP"`
  - `dashboard.custom_tools.hook: "Hook"`
  - `dashboard.custom_tools.mcp_note: "* 仅统计 mcp__ 前缀工具"`
  - `dashboard.custom_tools.none: "(none)"`
  - `dashboard.custom_tools.more: "... +%d more"`
  - `dashboard.custom_tools.loading: "计算中…"`
  - `dashboard.custom_tools.error: "统计失败"`
- [ ] en.yaml 新增对应英文翻译
- [ ] `go test ./internal/i18n/...` 通过（验证所有键在两个 locale 均存在）

## User Stories

无直接用户故事映射，支撑所有展示类 AC 的文本正确性。

## Implementation Notes

- 键名遵循现有 `dashboard.*` 命名约定
- `dashboard.custom_tools.more` 使用 `%d` 占位符，调用方用 `fmt.Sprintf` 格式化
- 此任务可与 2.1 并行执行
